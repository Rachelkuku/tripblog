# PRD.md  
## 여행블로그 마스터 _아리찌

---

## 1. 📌 제품 개요

**제품명:** 여행블로그 마스터 _아리찌  

네이버 블로그 SEO 구조를 기반으로  
여행/제품/티켓/투어 등 다양한 글을  
👉 상위노출 구조의 프롬프트로 자동 생성하는 웹 서비스

---

## 2. 🎯 목표

- 블로그 초보도 상위노출 구조로 글 작성 가능
- 글 작성 시간 80% 단축
- 체류시간 높은 구조 자동 생성
- 수익형 블로그 전환 가능

---

## 3. 👤 타겟 유저

- 네이버 블로그 초보자
- 체험단 / 제휴 마케터
- 여행 블로거
- 수익형 블로그 운영자

---

## 4. 🧩 핵심 기능

### 4.1 사용자 입력

- 지역
- 여행 유형 (국내 / 해외)
- 여행 기간
- 주요 키워드
- 추가 정보

---

### 4.2 카테고리 선택

- 숙소 후기
- 숙소 위치 강조
- 제품 추천
- 티켓 예약
- 입장권 정보
- eSIM 추천
- 투어 추천

---

### 4.3 글 스타일

- 후기형
- 정보형
- 구매유도형

---

### 4.4 글 길이

- 1500자
- 2000자 이상

---

### 4.5 SEO 강도

- 일반
- 강하게

---

### 4.6 결과 출력

- 제목 5개
- SEO 키워드
- 블로그 작성 프롬프트
- 이미지 키워드 (Unsplash)

---

## 5. 🧠 핵심 로직

### 공통 SEO 규칙

- 키워드 5~8회 반복
- 제목 키워드 2개 이상 포함
- 첫 문장 자연스러운 도입
- 사진 위치 최소 6개
- 후기형 자연스러운 문체

---

### 카테고리별 구조

#### 숙소
- 위치 / 객실 / 조식 / 주변 인프라

#### 숙소 위치
- 위치 / 접근성 / 동선 / 주변 시설

#### 제품
- 사용 계기 / 특징 / 후기 / 장단점

#### 티켓
- 가격 / 할인 / 예약 방법

#### 입장권
- 가격 / 운영시간 / 구매방법

#### eSIM
- 로밍 비교 / 설치 / 사용 후기

#### 투어
- 코스 / 시간 / 이동 / 가이드

## 6. 🖼 이미지 기능

예:

- tour travel

기능:
- 이미지 6개 출력
- URL 복사 기능

## 7. ✨ 차별화 기능

### 7.1 블로그형 도입문 자동 생성

- 이번 여행을 준비하면서 가장 고민했던 부분이었습니다.
- 여러 옵션을 비교하다가 선택하게 된 곳입니다.
- 여행 준비하면서 필요한 정보들을 정리했습니다.
- 실제 이용해보니 만족도가 높았습니다.

---

### 7.2 키워드 확장

- 제주 호텔 위치 좋은 곳
- 제주 가성비 숙소
